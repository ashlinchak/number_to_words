NumberToWords
===

Convert number to words with Go.

## Usage
Import this package into your Go application:

```go
import github.com/ashlinchak/number_to_words
```

And use it:

```go
number_to_words.IntToWords(1000)
// one thousand
```

## Specifications
* Supported language
    - English
* Supported type
    - int (integer)

## Run tests
```
$ go test
```
